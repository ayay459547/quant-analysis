# config-conventional 說明

| type     | directions |
| -------- | ---------- |
| feat     | 增加新功能 |
| fix      | 修復bug |
| docs     | 只改動了文檔相關的內容 |
| style    | 不影響代碼含義的改動，例如去掉空格、改變縮進、增刪分號 |
| build    | 構造工具的或者外部依賴的改動，例如webpack，npm |
| refactor | 代碼重構時使用 |
| revert   | 執行git revert打印的message |
| test     | 添加測試或者修改現有測試 |
| perf     | 提高性能的改動 |
| ci       | 與CI（持續集成服務）有關的改動 |
| chore    | 不修改src或者test的其餘修改，例如構建過程或輔助工具的變動 |
