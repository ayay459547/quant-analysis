# 📈 Stock-Tracker：台股個股分析與追蹤系統

## 📝 專案簡介

本專案是一個全方位的台股分析與追蹤平台（Side Project）。旨在幫助投資人快速掌握個股的財務體質、籌碼動態、產業題材與最新新聞。透過數據視覺化與自動化亮點/風險提示，降低投資研究的時間成本。

## ✨ 核心功能清單 (Features)

### 👤 會員系統與個人化功能

* **帳號登入/註冊**：支援一般信箱註冊與第三方（Google/GitHub）快速登入。
* **新手引導任務**：設計如「查詢 3 家公司」、「連續上站 2 天」、「加入追蹤股」等任務，提升使用者留存率。
* **我的追蹤 (Watchlist)**：允許使用者將感興趣的標的加入自選股清單，並設定價格或籌碼異動的推播通知。

### 🔍 個股綜合儀表板 (Dashboard)

* **即時報價面板**：顯示當日收盤價、漲跌幅、成交量等基本資訊。
* **關鍵指標摘要**：一目瞭然的財務快照，包含本益比 (PE)、殖利率、股價淨值比 (PB)、近四季 EPS、ROE 及近期營收 YoY。
* **公司業務簡介**：提供公司主要業務說明、產品營收佔比與未來展望。

### 📊 籌碼動態分析 (Chip Analysis)

* **多區間籌碼追蹤**：支援查詢 1日、1周、1月至 1年 等不同時間跨度的籌碼變化。
* **主力籌碼指標**：視覺化呈現「主力淨買賣超」、「籌碼集中度」與「買賣家數差」。
* **券商分點排行**：列出該區間內買超與賣超前 20 大的券商分點明細（如：國泰、摩根大通、元大等）。

### 💡 題材與投資健診 (Theme & Health Check)

* **AI 關聯題材分析**：標記個股所屬概念股（如：AI 伺服器、先進製程、CoWoS 先進封裝等），並說明受惠原因與產業鏈關係。
* **大數據投資亮點/風險**：
* 自動抓取財報數據，提示如「毛利率連續 3 季成長」、「當季 EPS 創近 8 季新高」等利多亮點。
* 透過歷史回測數據，顯示具備該亮點的股票在未來 90 天的歷史上漲機率。
* 警示潛在的財務或產業風險。



### 📰 新聞與市場動態 (News Aggregation)

* **產業新聞聚合器**：自動抓取並分類與該公司及所屬產業相關的財經新聞（如：被動元件、半導體導線架等產業要聞）。
* **個股重大訊息**：即時更新證交所的個股重大公告、營收發布與法說會動態。

## 🛠️ 技術堆疊 (Tech Stack) *[建議可依據您的實際使用的技術填寫]*

* **前端 (Frontend):** React.js / Next.js, Tailwind CSS, Recharts (圖表)
* **後端 (Backend):** Node.js / Python (FastAPI)
* **資料庫 (Database):** PostgreSQL / MongoDB
* **爬蟲與資料源 (Data Source):** Python (BeautifulSoup, Selenium), 證交所公開資訊觀測站 API

## 🚀 本地端啟動 (Getting Started)

1. Clone 專案：`git clone https://github.com/yourusername/stock-tracker.git`
2. 安裝套件：`npm install`
3. 設定環境變數：請複製 `.env.example` 並重新命名為 `.env`，填入對應的 API Key。
4. 啟動伺服器：`npm run dev`
